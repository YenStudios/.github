# 🩷 Yen Studios

> Desenvolvendo experiências para Minecraft, Discord e aplicações modernas.

A **Yen Studios** é um studio especializado no desenvolvimento de soluções para comunidades e servidores de jogos. Nosso foco é criar projetos modernos, eficientes e de alta qualidade, sempre priorizando desempenho, organização e uma excelente experiência para os usuários.

## 🚀 O que fazemos

* 🔌 Plugins para Minecraft (Paper, Spigot, Purpur, Folia e derivados)
* 🏗️ Sistemas personalizados para servidores
* 🎮 Builds e estruturas para Minecraft
* 🤖 Bots e aplicações para Discord
* 🌐 APIs REST e microsserviços
* 📚 Bibliotecas e ferramentas para desenvolvedores
* ⚡ Soluções sob medida para comunidades

## 💡 Nossa missão

Desenvolver soluções inovadoras que unam qualidade, desempenho e facilidade de uso, ajudando servidores e comunidades a crescerem com tecnologia confiável e moderna.

## 📂 Repositórios
